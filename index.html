Dark Room AI

Dark Room AI is an interactive scary story experience designed to immerse you in a mysterious narrative guided by an artificial intelligence. The project was created as a fun and engaging way to experiment with AI storytelling capabilities.

Features

  • Interactive Storytelling
  • Multiple Languages
  • Word-by-Word Display
  • User Choices

Changelog for v0.36

  • Enhanced GUI
  • Improved AI Responses
  • Faster Response Time

Motivation

I made this project because I was bored and wanted to explore the creative potential of AI in generating immersive, interactive stories. This project leverages advanced natural language processing to provide a dynamic and engaging experience for users.

How to Play

  1. Start the Game
  2. Read the Story
  3. Make Choices
  4. Follow the Narrative
  5. End or Restart

Installation

To run the Dark Room AI locally, follow these steps:

git clone https://github.com/notBubble/darkroom.git

Open the HTML File: Navigate to the project directory and open index.html in your web browser.

FreePT

Welcome to the FreePT project! This repository contains the code for a website (notbubble.github.io/freept) where you can use ChatGPT for free.

Features

  • Free Access
  • User-Friendly Interface
  • Open Source

Usage

To use FreePT, simply go to the following URL: notbubble.github.io/freept.

Installation

To run this project locally, follow these steps:

git clone https://github.com/notbubble/freept.git
cd freept

Open the index.html file in your preferred web browser.
